# FRAMEWORK
#### React TypeScript SCSS


## Installation

[Node.js](https://nodejs.org/) v 10.13.0
[NPM](https://www.npmjs.com/) v 6.4.1

#### For production mode
```bash
npm install
npm build
```
`http//localhost:3000`
_____________________

#### For frontend development mode


```bash
npm install
npm run dev
```
`http//localhost:3008`
